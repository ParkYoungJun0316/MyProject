# Unity 6.3 LTS 설정 기준

## 목적
- 이 스킬은 **Unity 6.3 LTS (6000.3.9f1)** 버전을 사용하는 프로젝트에서, 에이전트가 **설정/인스펙터/프로젝트 옵션** 관련 답변을 줄 때 항상 이 버전을 기준으로 생각하도록 강제한다.
- 버전 차이로 인해 메뉴 위치, 옵션 이름, 패키지 동작 등이 달라질 수 있으므로, 모든 안내는 기본적으로 **Unity 6.3 LTS 환경**을 전제로 한다.

## 사용 시점
- 사용자가 아래와 같이 요청할 때 이 스킬을 적용한다.
  - Unity 에디터 **설정(Preferences, Project Settings)** 변경 방법을 물을 때
  - **Inspector 설정, 컴포넌트 옵션, Tag/Layer, Physics, Input, Rendering, URP/HDRP, Build Settings** 등의 값을 수정/추가/삭제하는 방법을 물을 때
  - **Cinemachine, Input System, NavMesh** 등 유니티 패키지의 설정 화면을 어떻게 맞춰야 하는지 설명할 때
  - 특정 기능이 **어느 메뉴에 있는지, 어떤 토글/드롭다운을 눌러야 하는지** 단계별로 설명해야 할 때

## 핵심 지침

1. **항상 Unity 6.3 LTS 기준으로 설명**
   - 메뉴 경로, 버튼 이름, 옵션 이름을 안내할 때는 Unity 6.3 LTS (6000.3.9f1) 버전에서 실제로 보이는 형태를 기준으로 설명한다.
   - 과거 버전(예: 2021, 2022, 6000.0 등)에서 위치나 이름이 다른 옵션은, 필요 시
     - "**이 프로젝트는 Unity 6.3 LTS(6000.3.9f1)를 기준으로 합니다**"라고 먼저 명시한 뒤
     - "이전 버전에서는 메뉴 위치/이름이 다를 수 있다"고 덧붙인다.

2. **버전 차이 의심 시 명시적으로 언급**
   - Unity 매뉴얼/튜토리얼/블로그 예시가 **다른 메이저 버전**을 기준으로 하는 것처럼 보이면, 그대로 따라가지 않고:
     - "해당 자료는 다른 Unity 버전을 기준으로 할 수 있습니다."
     - "이 프로젝트 기준 Unity 6.3 LTS에서는 메뉴 경로/이름이 다음과 같이 다를 수 있습니다."라고 정정하고,
     - 가능한 한 6.3 LTS 기준 경로나 옵션 명칭을 제시한다.

3. **설정/인스펙터 변경 시 공통 체크**
   - 설정/인스펙터를 바꾸라고 안내할 때, 아래 항목을 짧게라도 함께 고려한다.
     - 이 옵션이 **Unity 6.3 LTS에서 여전히 유효한지** (deprecated 여부)
     - 동일 기능을 6.3 LTS에서 **대체하는 새로운 옵션/워크플로우**가 있는지
     - URP/HDRP/빌트인 렌더 파이프라인 간에 **동일한 동작이 보장되는지**

4. **모호할 때는 6.3 LTS 문맥으로 정리**
   - 어떤 옵션/메뉴 이름이 버전에 따라 다르게 보일 수 있을 때:
     - 가능한 한 **Unity 6.3 LTS에서 실제로 보일 법한 명칭**을 먼저 제시하고,
     - 필요하면 "(이전 버전에서는 X라고 표시될 수 있음)"처럼 부가 설명만 추가한다.

## 답변 형식 가이드

Unity 설정/인스펙터 관련 답변을 작성할 때는, 가능하면 아래 템플릿 흐름을 따른다.

1. **버전 기준 명시 (필요 시)**
   - "이 프로젝트는 Unity 6.3 LTS (6000.3.9f1)를 기준으로 설명합니다."

2. **메뉴 경로/옵션 안내**
   - Unity 6.3 LTS에서 보이는 실제 메뉴 흐름으로 적는다.
   - 예시 형식:
     - "상단 메뉴에서 `Edit` → `Project Settings...` → `Input System` 으로 이동합니다."
     - "Hierarchy에서 오브젝트를 선택한 뒤 `Inspector`에서 `Add Component`를 눌러 `Cinemachine Virtual Camera`를 추가합니다."

3. **버전 차이 주의(필요할 때만)**
   - 같은 기능이 튜토리얼/문서와 다르게 보일 수 있을 때만, 간단히:
     - "Unity 6.3 LTS 기준으로는 이 옵션이 `X` 탭 아래에 있고, 이름이 `Y`로 표기됩니다."처럼 정리한다.

## 예시

### 예시 1: Input System 설정 안내

- "이 프로젝트는 Unity 6.3 LTS (6000.3.9f1)를 기준으로 설명합니다."
- "상단 메뉴에서 `Edit` → `Project Settings...` → `Player` → `Other Settings` 로 이동해 `Active Input Handling` 옵션을 확인합니다."
- "Unity 6.3 LTS 기준으로 `Both` 나 `Input System Package (New)` 중 하나를 선택할 수 있으며, 이 프로젝트의 Player 컨트롤 구조에 맞게 값을 제안합니다."

### 예시 2: Inspector에서 컴포넌트 옵션 변경

- "이 프로젝트는 Unity 6.3 LTS (6000.3.9f1) 기준입니다."
- "Hierarchy에서 `Player` 오브젝트를 선택한 뒤 `Inspector`에서 `Player` 스크립트 컴포넌트를 찾습니다."
- "Unity 6.3 LTS 기준으로 노출된 `[SerializeField]` 필드들을 조정하는 방식으로 수치를 변경하도록 안내하고, 코드에 하드코딩하지 않도록 설명합니다."
